# 🤖 AURA PAY Bot

Telegram бот для пополнения и вывода средств из казино (1xBet, Melbet, 1win, Mostbet).

---

## 📋 Функционал

- 💰 **Пополнение** — выбор казино → ID счёта → банк → сумма → QR для оплаты
- 💸 **Вывод** — казино → банк → телефон → QR от банка → ID → код → заявка
- 🌐 **Язык** — Русский / Кыргызча / O'zbekcha
- 📖 **Инструкция** — пошаговый гайд
- 🔑 **Админ /qr** — загрузить QR-код для оплаты
- 📩 **Админ /cob** — отправить сообщение любому пользователю
- 🚫 **Блокировка** пользователей из админ-панели

---

## 🚀 Деплой на Railway

### 1. GitHub

1. Создай новый репозиторий на [github.com](https://github.com)
2. Загрузи все файлы этой папки
3. Команды:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ТВОЙusername/aurapay-bot.git
git push -u origin main
```

### 2. Railway

1. Зайди на [railway.app](https://railway.app)
2. **New Project** → **Deploy from GitHub Repo**
3. Выбери свой репозиторий
4. После деплоя зайди в **Variables** и добавь переменные:

| Переменная      | Значение                          |
|-----------------|-----------------------------------|
| `BOT_TOKEN`     | Токен от @BotFather               |
| `ADMIN_CHAT_ID` | ID чата для уведомлений (например `-1001234567890`) |
| `ADMIN_IDS`     | Твой Telegram ID (например `123456789`) |

5. Railway сам запустит бота ✅

## ⚙️ Локальный запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать .env из примера
cp .env.example .env
# Заполни .env своими данными

```

## 🔧 Переменные окружения

| Переменная      | Описание                                                    |
|-----------------|-------------------------------------------------------------|
| `BOT_TOKEN`     | Токен бота от @BotFather                                    |
| `ADMIN_CHAT_ID` | ID группы/канала куда приходят заявки (число, может быть отрицательным для групп) |
| `ADMIN_IDS`     | ID администраторов через запятую — у них есть доступ к `/qr` и `/cob` |

## 📱 Команды бота

### Обычные пользователи
| Команда / Кнопка | Описание |
|------------------|----------|
| `/start`         | Запустить бота |
| `💰 Пополнить`   | Начать пополнение |
| `💸 Вывести`     | Начать вывод |
| `📖 Инструкция`  | Инструкция |
| `🌐 Язык`        | Выбор языка |

### Администраторы
| Команда | Описание |
|---------|----------|
| `/qr`   | Загрузить QR-код для пополнения (бот попросит фото) |
| `/cob <chat_id> <текст>` | Отправить сообщение пользователю. Пример: `/cob 123456789 Ваша заявка одобрена!` |

---

## 📬 Уведомления в админ-чат

При каждой заявке в `ADMIN_CHAT_ID` приходит уведомление с:
- Именем и ID пользователя
- Казино и ID счёта
- Суммой / банком / телефоном / кодом
- Фото QR (для вывода)
- Кнопками: ✍️ Написать | 🚫 Заблокировать | ✅ Разблокировать

---

## 📝 Примечания

- Банки при пополнении показаны как **«Не доступно»** — нажатие выдаёт алерт
- QR при пополнении появляется после `/qr` команды от админа
- Анимация «Генерирую QR...» длится 1–3 сек (рандомно)
- Анимация «Проверяю код...» длится 1–2 сек (рандомно)
